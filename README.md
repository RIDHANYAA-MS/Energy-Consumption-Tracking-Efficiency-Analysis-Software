# Energy Consumption Tracking & Efficiency Analysis Software

This project analyzes energy consumption data to identify usage trends and
high-consumption periods using structured datasets.

The goal is to improve understanding of energy usage patterns through software logic.

## Features
- Reads energy consumption data from CSV files
- Calculates peak and average usage
- Identifies high-consumption periods
- Simple and maintainable code

## Technologies Used
- Python
- CSV data
- Git
